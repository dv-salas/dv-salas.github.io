6. Impresión 3D

Como imprimir un objeto con CURA

En primera instancia debemos guardar la pieza que deseamos imprimir desde Fusion 360, en la sección herramientas con el comando crear.

Una vez en CURA, tendremos que seleccionar la impresora, en este caso la Creality Ender-3

Chequear que este parámetro se encuentre en 1,75mm

Seleccionamos slice

Dentro de los materiales escogemos el PLA+, el color es a elección, en este caso morado, pero podría ser cualquiera

En layer heigh escogemos la altura de cada capa de impresión, esto mientras más bajo nos dará mayor definición, pero también tomará mucho más tiempo

Apretando Slice podemos revisar el tiempo que tomará la impresión

Podemos cambiar la velocidad por ejemplo a 60 mm/s

Optimización de Noozle

Se puede también cambiar el diametro del Nozzle, lo cual de manera inversamente proporcional servirá para jugar con la definición y el tiempo de impresión

Con el mismo fin anterior, existen varios parámetros que podemos ir moviendo que afectarán de manera notable el tiempo de impresión como el ta mencionado Layer Heigh, este último se puede apreciar en el siguiente cambio

Impresora 3D SLA

Para esto último usaremos otra impresora (Form 3), que en vez de PLA utiliza resina, además cambiaremos al software Preform

Le agregaremos los soportes necesarios para su impresión, esto se puede hacer de manera automática

A pesar de ser automático estos se pueden editar, para así ajustar los detalles como por ejemplo querer dejar alguna parte de nuestro objetos sin soportes para un acabado más liso, el programa irá actualizándose e informandonos si los soportes son los suficientes

Podemos revisar también los distintos momentos del proceso de impresión
