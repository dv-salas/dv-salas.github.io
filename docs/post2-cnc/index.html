2. CNC

Tipos de fresadoras

Las fresadoras son las máquinas ocupadas para producir las desbastaciones, cortes y/o perforaciones utilizadas en el proceso de CNC, en este sentido es un proceso que produce piezas a partir de la sustracción de material, a diferencia de los procesos aditivos como la impresión con resina plástica. Entre sus ventajas se encuentra la gran precisión que posee y el bajo coste para materiales técnicos vs otros métodos.

Existen diferentes fresadoras, que se pueden clasificar por diferentes parametros, orientación del husillo, cantidad de ejes, según estructura, puente móvil o pórtico, de columna, etc.

Tipos de fresas

Además de las fresadoras también existen diferentes tipos de fresas, estas son las herramientas de corte que emplean las máquinas.

Las fresas poseen uno o más filos, los cuales se hacen rotar a grandes velocidad para producir los cortes, perforaciones, etc.
