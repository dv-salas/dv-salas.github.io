1. Modelado 3D de Superficies

Este es un testeo de comandos para generar sólidos a partir de dibujos planes, o sketchs. En primer lugar nos vamos a superficies y creamos un boceto, seleccionamos el plano donde lo haremos.

Luego con el comando de spline generamos un plano cualquiera

Aplicamos un plano de desfase en el cual haremos una segunda figura con el mismo comando de la primera

Crearemos una solevación o loft opara unir ambas figuras y generar una superficie

Luego generaremos un segundo sketch pero de manera perpendicular al cual le aplicaremos una extrucción

En la sección de sólidos aplicaremos una división para dividir nuestra primera superficie con la segunda

Nuevamente en la sección de superficies aplicaremos recortar o trim para eliminar las superficies sobrantes

Borramos las superficies sobrantes

Con el comando patch o parche cerraremos nuestra superficie, tanto por abajo como por arriba

Luego con “coser” unimos todas las superficies

Volviendo a la sección de sólidos aplicamos unos fillets para suavizar los bordes del objeto

Aplicaremos un shell o vaciado, para que nuestra figura quede hueca, podemos especificar el grosor de estos nuevos muros

En inspeccionar, con análosis de sección podemos revisar este último cambio
